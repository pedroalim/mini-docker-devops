Conceitos Fundamentais do Docker

O que é um Container?

Um container é uma unidade padronizada de software que empacota código e todas as suas dependências para que a aplicação seja executada de forma rápida e confiável em diferentes ambientes computacionais.

Docker vs. Máquinas Virtuais

Diferente das máquinas virtuais tradicionais, containers Docker compartilham o mesmo kernel do sistema operacional e isolam os processos da aplicação do resto do sistema. Isso os torna mais leves e rápidos de iniciar.

Componentes Principais

  • Docker Engine: O motor que executa os containers
  • Dockerfile: Arquivo de configuração para criar imagens
  • Docker Image: Template para criar containers
  • Docker Container: Instância em execução de uma imagem
