Sobre Este Projeto

Contexto da Atividade "Mini Site Docker"

Este site foi desenvolvido como resultado prático da atividade acadêmica "Mini Site Docker". O objetivo central do projeto transcendeu a simples criação de páginas web, focando na aplicação de um fluxo de trabalho de DevOps moderno e completo.[1]

A tarefa envolveu a criação de um site estático informativo utilizando HTML e CSS, que foi então conteinerizado com Docker usando uma imagem Nginx. O ponto crucial da atividade foi a automação do processo de build através de um pipeline de Integração Contínua (CI/CD) com GitHub Actions, garantindo que cada alteração no código fosse testada automaticamente.[1]
